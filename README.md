## 🎮 프로젝트 소개

**우결 사다리게임**은 우리 결혼했어요(We Got Married) 스타일의 남녀 커플 매칭을 위한 사다리타기 웹 앱입니다.

방송 스트리밍이나 화면공유 환경에서 **주최자 혼자 운영**할 수 있도록 설계되었으며, 화면에 티가 나지 않게 특정 참가자를 사다리 결과에서 제외하는 **비밀 설정 기능**을 제공합니다.

<br/>

## ✨ 주요 기능

### 🎯 사다리 게임
- 전통적인 한국식 사다리타기 로직 (랜덤 선택 없이 전원 참여)
- 남자 전원이 사다리 위에, 여자 이름이 아래에 배치
- **남자 수 > 여자 수**인 경우, 남은 자리에 **"다음에"** 슬롯이 랜덤 배치됨
- 특정 이름 칩 클릭 → 해당 인원의 사다리 경로만 애니메이션
- **전체 사다리 타기!** 버튼 → 모든 인원 순차 애니메이션
- 경로가 완성될 때마다 결과가 하나씩 아래에 나타남

### 👥 참가자 관리
- 이름 + 성별(남/여)로 참가자 추가
- 이름 중복 방지 처리
- 참가자 삭제
- 남자/여자 별도 컬럼으로 분리 표시
- 실시간 인원 통계 (남 N명 💞 여 N명)

### 🔒 비밀 제외 기능 (화면공유 안전)
- 헤더의 ⚙️ 버튼 클릭 → 참가자 설정 모달 오픈
- 특정 참가자를 사다리에서 **조용히 제외** 가능
- 제외된 참가자는 목록에서 취소선으로만 표시 (눈에 띄는 "0%" 뱃지 없음)
- 모달 이름도 "참가자 설정"으로 중립적 → 화면공유 중 노출되어도 자연스러움
- ⚙️ 버튼에는 제외자가 있을 때만 빨간 점이 생김

### 🎨 파스텔 플랫 디자인
- 그라데이션 없는 단색 파스텔 팔레트
- 남자: 파스텔 스카이블루 / 여자: 파스텔 로즈핑크
- 모든 색상에 대비 높은 어두운 텍스트 색상 매칭 (접근성 고려)
- 부드러운 페이드업 / 스케일인 등 마이크로 애니메이션

## 🚀 설치 및 실행

### 사전 조건
- Node.js 18 이상
- npm 8 이상

### 로컬 실행

```bash
# 저장소 클론
git clone https://github.com/zyansuh/ggmarried.git
cd ggmarried

# 의존성 설치
npm install

# 개발 서버 시작
npm run dev
```

브라우저에서 [http://localhost:5173](http://localhost:5173) 접속

### 프로덕션 빌드

```bash
# 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

## 📖 사용 방법

### Step 1 — 참가자 추가
1. 성별 버튼(남자 / 여자)으로 성별 선택
2. 이름 입력 후 **Enter** 또는 **추가** 버튼 클릭
3. 원하는 만큼 반복해서 추가

### Step 2 — 비밀 제외 설정 (선택)
1. 헤더 오른쪽 **⚙️** 아이콘 클릭
2. 사다리에서 빠져야 할 참가자 체크
3. **확인** 클릭 (체크된 참가자는 게임에서 조용히 제외됨)

### Step 3 — 게임 시작
1. 남자 1명 이상 + 여자 1명 이상이면 **🎉 사다리 게임 시작!** 활성화
2. 버튼 클릭 → 게임 화면 이동

### Step 4 — 사다리 타기
- **이름 칩 클릭** → 해당 사람의 경로만 애니메이션으로 그려짐
- **🎲 전체 사다리 타기!** → 모든 인원 순서대로 자동 실행

### Step 5 — 결과 확인
- 경로가 완성될 때마다 아래 결과 칸에 커플이 하나씩 등장
- 여자 인원이 부족하면 일부 남자는 **"다음 기회에"** 슬롯에 도착

### Step 6 — 다시 돌리기
- **🔄 다시 돌리기** → 새로운 사다리와 슬롯 배치로 재시작
- **← 참가자 수정** → 참가자 화면으로 돌아가기

## 🧩 컴포넌트 상세

### `App.jsx`
- 화면 상태 관리: `'setup'` ↔ `'game'`
- 참가자 배열 CRUD: `addParticipant`, `removeParticipant`, `toggleExclude`
- 게임 화면에 남자/여자 배열 분리해서 전달

### `ParticipantSetup.jsx`
| 하위 컴포넌트 | 역할 |
|---|---|
| `SecretModal` | ⚙️ 비밀 제외 설정 모달 |
| `GenderList` | 남자/여자 참가자 컬럼 |

### `LadderGame.jsx`
| 순수 함수 | 역할 |
|---|---|
| `buildBridges(cols)` | 랜덤 가로 다리 생성 |
| `traverse(bridges, startCol)` | 한 열 사다리 타기 → 경로 포인트 반환 |
| `buildBottomSlots(females, cols)` | 여자 + "다음에" 슬롯 셔플 배치 |
| `shuffle(arr)` | Fisher-Yates 셔플 |
| `renderCanvas(...)` | HTML5 Canvas 렌더링 (레일, 다리, 경로 글로우) |
